nlp_suite.text_analysis.text_analysis.get_plots(message_lengths, average_word_lengths, stop_dic, unigrams, bigrams, trigrams, wordcloud)¶

Runs all plotting and saving functions, each in its own figure. All plots are saved in ‘plots’ folder

Parameters

  • message_lengths (pd.Series) – each message’s length in terms of words

  • average_word_length (pd.Series) – each message’s average word length

  • stop_dic (collections.defaultdict) – frequencies (dict values) for each stopword (dict keys)

  • unigrams (list of tuples) – decending ordered, by frequency, list of tuples: (unigram, frequency)

  • bigrams (list of tuples) – decending ordered by frequency list of tuples: (bigram, frequency)

  • trigrams (list of tuples) – decending ordered by frequency list of tuples: (trigram, frequency)

  • wordcloud (worldcloud.WorldCloud) – word cloud function, plot using plt.imshow
