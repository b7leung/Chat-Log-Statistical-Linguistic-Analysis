nlp_suite.clustering.utils.classify_text(vectorizer, clusters, text_docs)¶

Classify documents of text given a learned vectorizer and cluster centers

Parameters

  • vectorizer (sklearn.feature_extraction.text.TfidfVectorizer) – Vectorizer computed using sklearn used to encode text

  • clusters (sklearn.cluster._kmeans.MiniBatchKMeans) – K-Means clusters computed by sklearn

  • text_docs (collections.abc.Iterable) – Iterable containing text fragments to be classified

Returns

Numpy array of class predictions of the same length as text_docs

Return type

numpy.ndarray

nlp_suite.clustering.utils.plot_3d_clusters(pca_proj, labels, max_points=10000, point_size=2, theme='plotly_dark')¶

Plot 3D pca projections on a plotly graph

Parameters

  • pca_proj (numpy.ndarray) – Numpy array consisting of 3D projections for each data point; Shape [N, 3]

  • labels (numpy.ndarray) – Numpy array consisting of clustering label for each data point; Shape [N,]

  • max_points (int, optional) – Maximum number of points to plot, defaults to 10000

  • point_size (int, optional) – Size of plotted points, defaults to 2

  • theme (str, optional) – Plotly theme for generated plot, defaults to ‘plotly_dark’

Returns

Plotly FigureWidget object contatining a cluster plot

Return type

plotly.graph_objs._figurewidget.FigureWidget
